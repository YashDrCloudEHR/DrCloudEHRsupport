DrCloudEHR : 3. Scheduling Task

Created by Pankaj Bhagat on Aug 19, 2024

Hereâ€™s how you can easily schedule a task for yourself or assign it to someone else.

  1. Navigate to the tasks tab from the home page.
  2. Select "View All" to access the dropdown menu.
  3. Begin scheduling the task by entering the task information.
  4. The task will default to being assigned to yourself. If you wish to assign it to someone else, you can change this.
  5. Set the status of the task. Options include:
    1. Not Started
    2. In Progress
    3. Waiting on Someone Else
    4. Completed
    5. Deferred
  6. Enter a subject for the task. You can choose from predefined subjects or type a new one. For example, "Send a letter for recertification."
  7. Select contacts from your contact list if applicable. You can also add an additional person if needed.
  8. Choose a due date for the task. For instance, you might set it for one week from today.
  9. Set the priority level of the task as Normal, High, or Low. The default is Normal.
  10. Type in comments or a description of the task to provide more details.
  11. Save the task once all information is entered.
    To make changes to the task:
    1. Click on the pencil icon to edit the task details.
    2. You can mark the task as complete once finished.
    3. Edit comments, change the due date, or create a follow-up task if necessary.

After creating the task, you can view it at any time by returning to the home page and selecting "View All." You can also change the status to "In Progress" as you work on the task.
