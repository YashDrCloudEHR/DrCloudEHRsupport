DrCloudEHR : Using TeleHealth for Remote Treatment

Created by Dan Hynes on Aug 27, 2020

DrCloudEHR provides a TeleHealth feature through integration with the Zoom for Healthcare video conferencing platform. Providers can use the TeleHealth feature to meet remotely with patients without having to leave the DrCloudEHR application.

The application automatically creates a Zoom session when you check a patient in for a TeleHealth appointment or create a new encounter, and emails the patient the Zoom meeting details. When you close the encounter, the application adds the "GT" modifier to service codes in the corresponding fee sheet to facilitate billing.

Enable TeleHealth

Complete the following steps to enable your organization to use the TeleHealth feature. Note that both the provider and the patient need to install the Zoom application on their respective devices.

Add a TeleHealth Category to Use with Patient Appointments

You must create an encounter category to use with TeleHealth appointments and encounters. For more information, see see Create Categories.

When you create the category, select the following in the Create Category window:

  • Select the TeleHealth checkbox.
  • Select Patient from the Type menu. 
  • Select 02: Telehealth from the Place of Service menu.

Enable Patients to Receive Email Messages from DrCloudEHR

DrCloudEHR sends email to patients with the details for a TeleHealth meeting. Ensure that patients are able to receive emails sent by the application.

  1. Click the Patients tab, and then search for the patient.
  2. Click the Edit button next to Demographics on the Patient Summary Chart page.
  3. Click the Choices tab under Demographics.
  4. Set the Allow Email property to YES.

Enable a DrCloudEHR User to Use TeleHealth

You can enable a new or existing DrCloudEHR user to use the TeleHealth feature.

When you enable a user to use the feature, the application creates a user account in Zoom. The email address you enter for the user in the user's account profile is used as the user's Zoom account identifier. You must update the TeleHealth user account if the user's email address changes.

  1. Click the Administration tab.
  2. Click Users in the left navigation bar.
  3. Add a new user, or select an existing user.
  4. Click the TeleHealth tab in the Add User or Edit User popup window.
  5. Enter the configuration details.
  6. Create or update the user's Zoom account.
    • If you are adding a user, you choose to create the TeleHealth user after you click Save.
    • If you are editing a user, click the Create User at TeleHealth button.
    • If you modify the email address in the user's account profile, click the Update User at TeleHealth button to update the Zoom account with the new address.
  7. Click Save.

Schedule and Start a TeleHealth Session

Select the TeleHealth category when you schedule a TeleHealth appointment event. Selecting the category causes the application to create a Zoom meeting and send an email with the meeting details to both you and the patient. The TeleHealth Appointment icon is also displayed in the appointment event details.

When the you check a patient in for a TeleHealth appointment, a Start TeleHealth Meeting button is displayed in the Telehealth appointment. Click the button to start the Zoom session.

You can also click the Copy Meeting Link button to copy the link and paste it into a Web browser.

When you close the encounter, the application adds the "GT" modifier to service codes in the corresponding fee sheet.
