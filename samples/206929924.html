DrCloudEHR : 2. How to Edit a Person's Details

Created by Pankaj Bhagat, last modified on Aug 19, 2024

Updating a person's details in the system is a straightforward process. Follow these simple instructions to ensure that all necessary information is accurately recorded.

  1. Select the person you wish to update from your person's list and access their chart.
  2. Navigate to the details section where you will find small pencil icons indicating areas that can be edited.
  3. Click on a specific area to make changes. For instance, you might want to add an external ID or any other relevant number.
  4. Continue down the details section, as it will expand to allow for further updates.
  5. If needed, you can revisit each section to make additional edits. The interface will open up as if you were setting up a new person, but you will be in edit mode.
  6. Once you have completed all your edits, select the save option to ensure that your changes are recorded.
