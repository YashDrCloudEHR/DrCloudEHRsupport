DrCloudEHR : Visit Tools

Created by Kalyan Chakravartin on Sep 14, 2020

DrCloudEHR also provides a set of widgets/tools, that are called Visit Tools, which are used to display the information and provide a way for the user to enter data.

These widgets allows the user to display informaiton, such as service codes, units of service, diagnosis of the patient, allergies reported by the patient, medications and its usage and more.

The following are the visit tools:

  • Visit Services
  • Client Diagnosis
  • Client Allergies
  • Client Medications
  • Treatment Plan Data (MDTP)
  • Form Builder Data Import
  • Existing Data Column
  • Demographics Data
  • Create Multi Rows

Visit Services

Visit services, generally known as Service Widget. Displays a list of services provided for the patients visit along with service codes. It also allow the user to enter new service codes. 

Client Diagnosis

Displays a list of patient diagnosis, which can be determined by the current provider, or may be based on previous diagnosis, where the patient had previously been seen.

Client Allergies

Displays a list of allergies reported by the patient.

Client Medications

Displays a lit of medications being taken by the patient as prescribed by the doctor treating the patient, using DrCloudEHR.

Treatment Plan Data (MDTP)

Treatment plan data field allows the user to use multi discipline treatment plan, if enabled, can be used in coordinated services and therapies for a specific patient.

Form Builder Data Import

Form builder data import allows the user to import a specific section of data from previously saved form builder forms. 

Existing Data Column

Existing data column field allows the user to access patients and providers database tables for displaying the required data in the form.

Demographics Data

Demographics data field allows the user to display data stored in the patients demographics page.

Create Multi Rows

Multi row field allows the user to create a multi row table with maximum five column.
