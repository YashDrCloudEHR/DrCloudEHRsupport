DrCloudEHR : Using Laboratory Interfaces

Created by Dan Hynes on Sep 16, 2020

DrCloudEHR supports integrated laboratory interfaces that allow you to interact directly with laboratories from within the application. 

The application supports two types of interfaces:

  • Results-only interfaces allow you to download and display laboratory test results. 
  • Results and order interfaces allow you to submit procedure orders electronically, and then download and display laboratory test results.

Results-only interfaces are available for the following laboratories:

  • Clinical Science Labs
  • Cordant Health Solutions
  • Kashi Clinical Laboratories
  • Millennium Health
  • Sterling Laboratories
  • Westox Labs

Results and order interfaces are available for the following laboratories:

  • LabCorp
  • Quest Diagnostics (Note: You must work directly with Williams Medical Technologies (www.williamsmedtech.com) to integrate and use the Quest Diagnostics interface.)

The application and the laboratory use the Health Level Seven (HL7) standard to transfer data. DrCloudEHR automatically downloads the HL7 results from the laboratory on a daily schedule, and matches the results with the corresponding patient. The application notifies the provider who ordered the tests that the results have been downloaded, and enables the ordering provider to review and sign off on the results.
