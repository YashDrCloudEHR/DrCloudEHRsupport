DrCloudEHR : 3. Admin- Reset User Password

Created by Pankaj Bhagat, last modified on Aug 26, 2024


Watch Video

Resetting a user's password in the system is a straightforward process. Follow these instructions to ensure the user can regain access to their account.

  1. Navigate to the upper right corner of the screen and select the gears icon for setup.
  2. Click on "Setup" again from the dropdown menu.
  3. Scroll down to the "Users" section and select "Users."
  4. Locate the user whose password needs to be reset.
  5. Check the box next to the user's name.
  6. Click on "Reset Password." This action will trigger an email to the user, allowing them to verify their identity and set up a new password.
  7. Repeat the process by checking the box next to the user's name and selecting "Reset Password" once more if necessary.

By following these steps, the user will receive the necessary instructions to create a new password for their account.
