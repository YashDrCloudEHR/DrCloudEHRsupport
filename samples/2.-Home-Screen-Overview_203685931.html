DrCloudEHR : 2. Home Screen Overview

Created by Pankaj Bhagat, last modified on Aug 06, 2024

Let’s get an overview of the home page of the logged in user. The home screen is personalized for each user, displaying user-specific information.

VIDEO: Home Screen for IDD

The Home Screen layout has 4 primary sections.

  1. Today’s Agenda
    1. Shows appointments scheduled for the user, including details like start date, time, subject, linked person, and assignee.
    2. Users can create new agenda or calendar items.
  2. House Lists
    1. Lists houses the user is assigned to work in.
  3. My Tasks
    1. Displays tasks assigned to the user, which can be self-assigned or assigned by others.
  4. Recently Viewed Persons
    1. Displays view of persons the user has recently viewed.
