DrCloudEHR : 8. How to Enter an Immunization Record

Created by Pankaj Bhagat, last modified on Aug 19, 2024

 Follow the steps below to accurately input immunization data:

  1. Access the patient’s chart. In this example (as seen in video), the patient is Jazzy Jones.
  2. Locate the immunization section. If it is not displayed, check the drop-down menu and select "New."
  3. Choose "Immunization" as the type.
  4. Select the appropriate code for the immunization being administered. The codes should already be set up in the system.
  5. Enter the lot number for the vaccine. If it has been used for other vaccines, it may already be in the system; otherwise, you will need to input it.
  6. Input the action date, which is the date the vaccine is being given.
  7. Specify the vaccine manufacturer.
  8. Enter the expiration date of the vaccine.
  9. Select the vaccine site where the immunization is administered.
  10. Add any relevant notes or comments regarding the immunization.
  11. Click "save." If prompted, enter an approved rate. This information may already be in the system, but if not, you will need to provide it.
  12. After saving, check the immunization section to confirm that the immunization has been recorded. In this example, a Covid immunization is displayed.
  13. If the immunization section is not visible on the top bar, it can be found under "More options." Select "New" to fill in the immunization information.

By following these steps, you can successfully enter an immunization into the EHR.
