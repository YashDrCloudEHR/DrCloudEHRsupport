IT Security and Technology Policies

Password Requirements
All employees must use strong passwords with a minimum of 12 characters including uppercase, lowercase, numbers, and special characters. Passwords must be changed every 90 days and cannot be reused for the last 5 passwords. Multi-factor authentication (MFA) is required for all company accounts.

Device Management
Company-issued laptops and mobile devices must be encrypted and have security software installed. Personal devices used for work must be approved by IT and meet security standards. All devices must have screen lock enabled with automatic lock after 5 minutes of inactivity.

Data Protection
Sensitive company data must be stored only on approved cloud services or company servers. Never store sensitive data on personal devices or unapproved cloud storage. All data transfers must use encrypted channels. Report any suspected data breaches immediately to IT security.

Software and Applications
Only approved software can be installed on company devices. Employees must request approval from IT before installing new applications. Open-source software requires security review. Pirated or unlicensed software is strictly prohibited.

Email and Communication
Company email should be used for business purposes only. Be cautious of phishing emails and never click suspicious links or download attachments from unknown senders. Report suspicious emails to IT immediately. All email communications are subject to company monitoring for security purposes.

Remote Access
VPN access is required for remote connections to company systems. Employees must use company-approved VPN clients. Never access company systems from public Wi-Fi without VPN. All remote access sessions are logged and monitored.
