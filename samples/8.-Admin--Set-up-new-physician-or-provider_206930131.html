DrCloudEHR : 8. Admin- Set up new physician or provider

Created by Pankaj Bhagat, last modified on Aug 26, 2024

Watch Video

When setting up a new position, follow these instructions to ensure all necessary information is correctly entered.

  1. Navigate to the position tab in the system.
  2. Select the option to create a new position.
  3. Enter the name of the position in the designated field.
  4. Add the phone number of the individual associated with this position.
  5. Input their email address.
  6. Provide the address of the individual.
  7. Choose a specialization from the available options, such as neurologist or podiatrist.
  8. This selection will allow the physician to appear when scheduling appointments, particularly for relevant types of appointments like nutrition.
  9. Once all information is entered, confirm that the physician is now set up in your listing.
