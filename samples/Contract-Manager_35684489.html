The Contract Manager feature is designed to serve as a contract repository in which DrCloudEHR client organizations may record and manage contracts with multiple payers for single or multiple locations. Creating contracts allow providers to define payer specific allowable payment amounts for services (including specific code and modifier combinations), as defined in that provider’s contract/fee schedule. While having this information available within the DrCloudEHR system is a very valuable reference tool, there are also more useful applications and benefits of using Contract Manager (payment posting, claims follow up, accounts receivable management, etc.).

When posting insurance payments, whether manually or electronically (ERA processing), billing staff have the real time capability to see and flag items that were not paid according to your contract. This allows for more immediate claims follow up, knowledge of potential payer fee changes, and overall better Accounts Receivable Management. DrCloudEHR also provides reports that allow you to analyze claims payments that may be incorrect, based on allowable amounts defined using Contract Manager.

The DrCloudEHR team is currently developing an incredibly valuable enhancement to the Contract Manager. This soon to be released enhancement will allow our clients, if they so choose, to have DrCloudEHR automatically adjust the amount above the payer allowable amount (the difference between your Standard Fee and the payer Allowable payment amount). If enabled, this new Contract Manager function will automatically post a “contractual adjustment” to the encounter/invoice (assigning a client defined reason for the adjustment). In other words, the claim would be submitted with your Standard Fee amount, while the accounts receivable balance would then reflect the amount a provider can “expect” to be paid once the payer/patient responsible amounts have been satisfied. This benefits billers and payment posting staff (saves them the time/effort of posting adjustments), but also provides more realistic accounts receivable balances, allowing for better financial analysis and planning.

It is best to create payer specific Billing Rules, if applicable, BEFORE creating a payer Contract (see Billing Rules Guidance for information on creating Billing Rules.

Enable Contract Manager

To enable the Contract Manager, do the following:

  1. Click the Administration tab.
  2. On the left navigation menu, click Global Settings.
  3. On the Billing tab
  4. Select the Enable Contract Manager checkbox.
  5. Click Save.

Create New Contracts

To access the Contract Manager, you must first access Practice Settings by clicking on Practice (top menu), Administrative (left menu), then Practice (left menu). If the Insurance Companies list does not display, you may choose it by clicking Insurance Companies from the menu at the top of the Practice Settings page. To utilize Contract Manager to create a New Payer Contract, perform the following steps:

  1. Locate that payer in the list of Insurance Companies.
  2. Then click on the Manage button to the far right of that Insurance Company, in the column labeled Contracts
  3. Upon clicking the Manage button, the Contracts/Fee Schedule screen for that specific payer is displayed.
  4. Click the Add button to define a New Contract for this Insurance Company. The Add Contract/Fee Schedule screen displays the basic Contract information section
  5. Complete the three sections provided to define this contract’s basic information
    • Name: Enter a name for this contract. The name should represent the Payer and identify anything unique about this contract (to distinguish it from any other contract you may have with this payer)
    • Effective: Using the calendar icon provided, enter the Effective date of this contract into the “From” box provided. If the contract has an expiration date, enter that date in the “To” box. If there is no expiration date stipulated in your contract with this payer, leave the “To” field empty.
    • Select Facilities: From the box on the left, click to select any/all facilities to which this contract applies (you may select multiple facilities from the list by holding down the CTRL key on your computer’s keyboard as you select additional facilities). After selecting the facility(ies), use the arrow buttons to move the selected facilities to the box on the right. NOTE: Only Facilities which have been added to your DrCloudEHR system will show in the box on the left.
  6. Next, click on the Contract Details button at the top of the Add Contract/Fee Schedule page.
  7. This page allows you to add code specific allowable payment amounts, as defined in your contract with this particular payer. Using the Fee Schedule provided with your contract (you may have to request this information from the payer or obtain it from their website), add each Service Code and any applicable modifiers, along with the Contract Allowable amount for that code/modifier combination. To add the Service Code, perform the following steps:
    1. Click on the Select Service Code picklist drop down menu provided under the Service Code column of the Contract Details grid, and select a code you wish to add to this contract
    2. If the payer requires modifiers to be appended to the selected code for specifically defined criteria, add those modifiers in the Modifier column. Use separate rows to define multiple Service Code/Modifier combinations for the same service code. NOTE: If multiple modifiers are required for a single service code, enter all modifiers together in the Modifier field, separating each of them with a single space.
    3. Enter the Contract Allowable amount for a single unit of the selected code in the Contract Amount column.
    4. Click the Add Item button to add additional lines required to define all service codes in a given contract.
    5. When all service codes in the contract have been added and Contract Amounts entered, click the Save button at the top of the page to save the New Contract for the selected payer.

Helpful Hints

  1. If you have already defined payer specific service code/modifier combinations using the Billing Rules feature, you may choose to click the Import All Service Rule Codes button. This button displays the list of code/modifier combinations already created, allowing you to select any/all codes at once to import into this contract. This can save an enormous amount of time and effort required in adding each and every service code individually.



  2. If you have already defined a payer specific Contract that includes the same service codes as this new contract, you may choose to click the Import Existing Contract button. This button displays
    Select Contract: window, allowing you to select an existing contract from a picklist drop down menu and import ALL the codes from that contract into this new contract by simply clicking the
    Import button: Once imported, any of the imported data may be edited/deleted, as necessary. Using the Import feature can save an enormous amount of time and effort required in adding each and every service code individually and repeatedly for contracts that are either identical or very similar.

Editing An Existing Contract

The process for Editing an existing Contract is much like the process for adding New Contracts. From the Practice Settings page (Practice/Administrative/Practice), locate the Insurance Company, whose contract you wish to edit and click the Manage button in the Contracts column. From the Contracts/Fee Schedule page, click the Edit button to the left of the Existing Contract you wish to edit.

The Edit Contract/Fee Schedule page displays, allowing you to make any necessary changes in the Contract and/or Contract details sections. Once all edits are complete, simply click the Save button to lock in your changes.
