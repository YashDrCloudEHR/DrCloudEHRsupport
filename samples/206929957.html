DrCloudEHR : 9. Adding a Diagnosis to a Patient's Chart

Created by Pankaj Bhagat, last modified on Aug 19, 2024

To add a diagnosis to a person's chart, follow these instructions:

  1. From the person's chart tab, navigate to the services section and select diagnosis, then choose the option for new.
  2. Select the code type. Options include diagnosis, immunization, or service code type. In this case, select diagnosis.
  3. Determine whether the diagnosis is related to mental behavior or medical issues and enter the appropriate code. For this example (in the video), the code entered is F78.
  4. Specify a start date for the diagnosis. This can be the current date or a date from the past if the diagnosis was made earlier.
  5. Set the status of the diagnosis to approved.
  6. Ensure that the diagnosis is marked as an active diagnosis.
  7. Save the changes.

After completing these actions, you can return to the services and diagnosis section to verify that the diagnosis has been successfully added to the patient's chart.
