DrCloudEHR : Using Speech Dictation to Complete Forms

Created by Dan Hynes on Sep 11, 2020

DrCloudEHR integrates with the Nuance Dragon dictation service to support voice to text dictation for certain forms. With this feature, you can dictate notes that are added to the fields in a form as you work with a patient.

You can use the Dragon dictation service with any form built using the Form Builder tool. You can also use the feature with the following standard forms:

  • SOAP (subjective, objective, assessment and plan) notes
  • History and Physical Examination
  • Physician Discharge Summary
  • Psychiatric Evaluation
  • Psychiatric Progress Note
  • Medical Progress Note
  • XOOM Chart

When you open a form that supports dictation, the Nuance Dragon logo  and a microphone icon appear at the bottom of the page. The Nuance Dragon icon also appears in each field in the form that supports dictation.

Complete an Encounter Form Using Dictation

You must create or open an encounter to use the dictation feature. The forms you complete using dictation are attached to the encounter.

You can manually edit the text in a field you complete using dictation.

Ensure that your microphone is enabled before you begin.

  1. Click the Patients tab. 
  2. Click Patient Search/Add in the left hand navigation bar, and then search for the patient. 
  3. Select the patient from the search results.
    The Patient Summary Chart page opens.
  4. Create or open an encounter.  
  5. Select the form you want to fill out.
  6. Click the microphone icon.
    The icon changes from red to green. The field you are entering text to also appears green.
  7. Begin speaking to add notations to the first Dragon-enabled field in the form.
    Use the following voice commands to control your dictation:
    • "Full Stop": End a sentence and start a new one.
    • "Next Field": Move to the next field in the form.
  8. Save the form when you are done with dictation.
