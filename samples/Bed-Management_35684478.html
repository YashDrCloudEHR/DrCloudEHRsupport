Enable Bed Management

To enable bed management, do the following:

  1. Log on as an Administrator.
  2. Click the Administration tab.
  3. On the left navigation menu, click Global Settings. The Edit Global Settings page opens.
  4. Click the Features tab.
  5. Select the Enable Bed Management System check box.
  6. Click Save.

The Global Settings Page > Edit Bed Management

Add Building

To add a building:

  1. Click the Practice tab.
  2. On the left navigation menu, click Administrative > Buildings. The Buildings page opens.
  3. Click Add. An Add Building dialog box opens.
  4. Select the facility and enter a building code and name.
  5. Click Save.

The Buildings Page

Edit Building Details

To edit a building, do the following:

  1. In the Buildings page, click the building code link. The Edit Building dialog box opens.
  2. Edit the information that you want to change, and then click Save.

The Buildings Page

Add Floors

To add a floor, do the following:

  1. Click the Practice tab.
  2. On the left navigation menu, click Administrative > Floors. The Floors page opens.
  3. Click Add. An Add Floor dialog box opens.
  4. Select the facility, the building, and enter a floor code and floor name.
  5. Click Save.

The Floors Page

Add Rooms

To add a room, do the following:

  1. Click the Practice tab.
  2. On the left navigation menu, click Administrative > Rooms. The Rooms page opens.
  3. Click Add. An Add Room dialog box opens.
  4. Select the facility, building, floor, and type, and enter a room number.
  5. Click Save.

The Rooms Page

Add Beds

To add a bed, do the following:

  1. Click the Practice tab.
  2. On the left navigation menu, click Administrative > Beds. The Beds page opens.
  3. Click Add. An Add Bed dialog box opens.
  4. Select the facility, building, floor, room number, type, and gender. Also, add a bed number and select a status.
  5. Click Save Changes.

The Beds Page

Check Room Availability

To check room availability, do the following:

  1. Log on as an Admission user.
  2. Click the Practice tab.
  3. On the left navigation menu, click Administrative > Check Room Availability.
    The Check Room Availability page opens.
  4. Search for available rooms by facility, building, floor, bed type, and date.
  5. Click Check to check for available rooms.
    Available rooms display. You can click the Details link to view room details.

The Check Room Availability Page

Bed Booking

  1. Bed Booking is done clicking on the Book hyperlink and it results in opening bed booking dialog box.
  2. Tentative discharge date can be entered if available and clicked on Book button.
  3. Click on Book button , gives a message box as 'Confirm this bed?' with Ok and Cancel options.
  4. Click on Ok to confirm bed to the resident.
