DrCloudEHR : Tracking Methadone Treatment

Created by Dan Hynes on Sep 18, 2020

DrCloudEHR integrates with the Methasoft Treatment Management System to enable you to track methadone doses dispensed to patients in the treatment of opiate addiction.

The methadone dosage information is provided in a comma-separated value (.csv) file that you upload to DrCloudEHR. The application parses the file and uploads the information for each patient to the application database. 

Process a Methasoft File

  1. Click the Practice tab.
  2. Click Interface in the left navigation bar, and then click Methasoft.
  3. Click Choose File. The Upload File window opens.
  4. Click Choose File, and then navigate to the .csv file.
  5. Click Upload File.
  6. Click Process File.
