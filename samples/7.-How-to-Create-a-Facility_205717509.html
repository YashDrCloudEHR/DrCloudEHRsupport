How to Create a Facility

VIDEO: Create a Facility

Step-by-Step Process:

  1. Navigate to the Facilities Tab:
    • When creating a facility such as a house or a clinic, you will need to go to the facilities tab.
    • Select New: Click on the 'New' button to start the process.
  2. Choose Record Type:
    1. You will be prompted to choose the type of facility record. Options include clinic, funeral home, house, or pharmacy. For this example, select 'House'.
  3. Start Creating the Record:
    1. This action will open up the information form where you can begin entering details for the new facility.
  4. Enter Facility Name:
    1. Fill in the facility name in the designated field.
    2. Provide a Short Name: Optionally, you can give the facility a short name for easier reference.
  5. Associate with Main Agency:
    1. If applicable, tie the facility to your main agency.
  6. Set Active Status:
    1. Ensure you select the facility as 'Active' if it is currently operational.
  7. Record Facility Capacity:
    1. Enter the capacity details, including the number of rooms available, the number of beds, how many are already consumed, and how many are scheduled.
    2. Enter Address and Contact Information:
  8. Fill in the address of the home and the phone number.
    1. Optionally, add a fax number and email address.
  9. Fill Additional Information:
    1. Provide other relevant details such as taxonomy code and MPI (Master Patient Index) assignment.
  10. Save the Record:
    1. Once all information has been completed, click on 'Save'.
    2. Review the New Facility:

The new facility should now be listed under facilities, confirming its creation.
