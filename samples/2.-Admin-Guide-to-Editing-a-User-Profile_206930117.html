DrCloudEHR : 2. Admin Guide to Editing a User Profile

Created by Pankaj Bhagat, last modified on Aug 26, 2024

Watch Video

Editing a user profile is a straightforward process that can be accomplished through the user setup page. Follow the instructions below to make necessary updates to a user profile.

  1. Navigate to the user setup page to locate the user you wish to edit.
  2. Select the user from the list. For example, if you choose "user test," click on the edit option.
  3. Make the required changes to the user profile. For instance, if you need to change the user's role, you can update it to "DSP."
  4. Keep the user license unchanged unless a modification is necessary.
  5. In the profile section, select a organization base profile for DSP. This profile has different privileges and permissions compared to the previous role, such as a registered nurse.
  6. After making all necessary changes, click on the save button to apply the updates. 
  7. Following these instructions will ensure that the user profile is accurately updated to reflect the desired changes.
