DrCloudEHR : 1. Adding Goals & Progress

Created by Pankaj Bhagat, last modified on Aug 26, 2024

Watch Video

In this guide, we will outline the process of setting up goals in progress within a person's chart.

  1. Access the Goals Section
    Begin by navigating to the "more" dropdown menu in the person's chart. From there, select "goals in progress."
  2. Review Existing Goals
    You will see any existing goals, such as "healthy living," along with a brief description.
  3. Create a New Goal
    To add a new goal, click on the "new" option. A dropdown menu will appear, displaying various goals to choose from.
  4. Select a Goal
    Browse through the available goals and select one that fits your needs.
  5. Set the Start Date
    Enter the start date for the new goal. Ensure that it is assigned to the correct person's chart.
  6. Define Progress Steps
    Input the specific actions or steps you plan to take to progress on this goal.
  7. Check Completion Status
    Determine whether the goal has been completed. In this case, it has not been completed as work is still ongoing.
  8. Save the Goal
    Finally, save the newly added goal to the goals in progress section.

By following these steps, you can effectively set up and manage goals in progress within a person's chart.
