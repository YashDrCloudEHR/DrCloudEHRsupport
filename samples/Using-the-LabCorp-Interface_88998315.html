DrCloudEHR : Using the LabCorp Interface

Created by Dan Hynes on Sep 18, 2020

DrCloudEHR provides an interface that allows you to submit procedure orders electronically to LabCorp, and then download and display laboratory test results. 

After a requested test is completed, DrCloudEHR automatically downloads the results from the laboratory and matches the results with the patient for which the tests were done. The application notifies the provider who ordered the tests and enables the provider to review and sign off on the results.

Download and View Results

DrCloudEHR automatically and securely downloads results from the laboratory and uploads the results to the database on a set schedule. The application parses each result and matches it with the corresponding patient  based on the order requisition number.

When a result is matched with a patient, the application does the following:

  • Uploads the results to the patient's chart. You can view the results in HTML format by clicking the Practice > Procedures > Electronic Reports page.
  • Saves the results as a PDF file in the patient's Documents > LabCorp folder.
  • Sends a notification to the provider who ordered the test, and to the user group specified in the lab connection. 
  • Creates an encounter and attaches the results to it.

If the application is unable to match a result with a patient, you can perform the match manually.

View Orders and Results

You can download test results for matched patients formatted as PDF files on the LabCorp Orders/Results page. You can also view procedure orders sent to the lab.

The page displays two results download links for each procedure. The LabCorp results displays the results as delivered from LabCorp. The DrCloud results contains the same results data in the format used by the application.

  1. Click the Practice tab.
  2. Click Procedures in the left navigation bar, and then click LabCorp Orders/Results.
  3. Enter the search criteria, and then click Search.
  4. Click an order or result link to view the data in PDF format.

Match Results and Patients Manually

The Unmatched Results button displays the number of results that you must match manually.

  1. Click the Practice tab.
  2. Click Procedures in the left navigation bar, and then click LabCorp Orders/Results.
  3. Enter the search criteria, and then click Search.
  4. Click the Unmatched Results button to display unmatched results.
  5. Click the Unmatched Results tab.
  6. Click the link for each unmatched patient in the Patient Name column. The application displays a list of suggested patients.
  7. Click the Match Patient Manually tab.
  8. Start typing patient's name in Patient field, then select the patient to match the results with.
  9. Click the Match Patient button.

View Results in the Patient Documents Folder

When the application downloads lab test results, the results are saved as a PDF file in the patient's Documents > LabCorp Results folder.

  1. Click the Patients tab, and then select for the patient for whom you want to download the results file.
  2. Mouse over the Documents tab at the top of the Patient Summary Chart page, and then select Documents. The folder tree displays.
  3. Expand the LabCorp Results folder in the tree.
  4. Click the PDF file.

View Results in HTML Format

You can generate a report displaying the test results for a patient in HTML format.

  1. Click the Patients tab, and then select the patient for whom you want to view results. The Patient Summary Chart page displays.
  2. Click the Practice tab.
  3. Click Procedures in the left navigation bar, and then click Patient Results. 
    Note that you must select or create an encounter to enable the tab.
  4. Select the encounter the test results are saved to from the Lab Procedure Encounter menu.
  5. Click Generate Report to view the results in HTML format in a new tab in your web browser.

Review and Sign Off On Results as a Provider

As a provider, you can review the lab results report for a patient. After you review the results, you electronically sign the report. The application then moves the report to the Patient Results page for the patient.

  1. Click the Patients tab, and then select the patient for whom you want to view results. The Patient Summary Chart page displays.
  2. Click the Practice tab.
  3. Click Procedures in the left navigation bar, and then click Pending Review.
    Note that you must select or create an encounter to enable the tab.
  4. Click Generate Report to view the results in HTML format in a new tab in your web browser.
  5. Return to the pending review page and select for each result from the Status menu.
  6. Select a value from the Abn menu for each result report to indicate whether the results are abnormal.
  7. After you review the report, click Sign Results.
    The report is added to the Patient Results page, which contains a list of all lab result reports for the patient.

Create and Upload a Procedure Order to LabCorp

The LabCorp interface allows you to submit procedure orders electronically from DrCloudEHR. You must first select the patient and then create or open an encounter to submit a procedure order. 

  1. Click the Patients tab, and then search for the patient .
  2. Open or create an encounter for the patient.
  3. Click Forms Cabinet in the left navigation bar, and then click General.
  4. Click the Provider Order (Lab Corp) form. 
    Note that you must select or create an encounter to enable the form.
  5. Provide the order details.
  6. Select the payer from the Bill Type menu.
    If you select Third Party, select the LabCorp special billing type from the Select Special Bill menu. You must also click the Add button next to Diagnosis at the bottom of the page, and then click in the field to add a diagnosis code. 
  7. If an advanced beneficiary notice is required, select the ABN Required checkbox, and then select the courtesy copy type.
  8. Click the Add button next to Specimen, Diagnosis, and Order, and then select the appropriate values.
  9. Select the checkbox to electronically sign the order if required.
  10. Click Save. The order is saved to the encounter.
  11. Click the Edit button above LabCorp order on the encounter page.
  12. Review the order. If it appears correct, click Transmit to submit the order to LabCorp.
    Note that you can also click the E-Requisition button to generate a .zip file containing a PDF version of the order and a text file containing a bar code number.
