Comments:

In my view;

One accesses a Module using a Tab.

e.g. Click on the Calendar tab to access the Calendar module.

It would be inappropriate to describe the collection of sub pages and funtions as contained inside a tab.

It would also be inappropriate to use the word Module to describe the rectangular top link we click on when describing navigation.

Navigation: Tab->Side-Menu->Sub-Menu

Content: Module->Page→Dialog
