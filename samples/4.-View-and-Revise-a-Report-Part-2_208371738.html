DrCloudEHR : 4. View and Revise a Report Part 2

Created by Pankaj Bhagat on Aug 27, 2024

Hereâ€™s a concise guide on how to view, export, and edit reports effectively.

  1. Navigate to the main report screen. Each report has a dropdown menu located on the far right.
  2. Click on the dropdown menu. You will see several options: run the report, edit the report, subscribe to the report, export it, delete it, add it to a dashboard, mark it as a favorite, or move it.
  3. To export a report, select the export option. You will be prompted to choose a formatting style. Options may include different formats; for this example, select the Excel format.
  4. After selecting the format, click on export. The report will download to your computer.
  5. Open the downloaded file. You may find that it contains minimal information. You can enable editing to modify the report.
  6. To edit the report, return to the dropdown menu and select the edit option. This will allow you to add additional information to the report.
  7. If you want to include specific data, such as contacts, race, or gender, you can do so in the editing mode. For instance, add the race information to the report.
  8. Similarly, you can add gender or sex information. This will create a customized report with additional columns.
  9. After making the desired changes, you can run the report to view the results. This will not affect the original report.
  10. To confirm, navigate back to the reports section and select the original report. It will remain unchanged, without the additional information you added.
