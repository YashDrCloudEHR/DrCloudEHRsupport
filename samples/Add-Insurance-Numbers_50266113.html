DrCloudEHR : Add Insurance Numbers

Created by Tejasvini Aurangabadkar on Jun 07, 2018

Insurance Numbers are the numbers used as Provider's unique identity numbers in the Claim forms. These unique identity numbers retrieves the insurance numbers  You can add  insurance number to the existing providers.


To add the Insurance Numbers, perform the following steps:

  1. Navigate to Practice > Administrative > Practice.
  2. Click Insurance Numbers in the Practice Settings section. The Insurance Numbers list appears:
  3. Click the required name of the user from the Insurance Numbers list.
  4. Click Add New
  5. Enter the required information in the following Add Provider Number section:
    • Insurance Company: Specifies the name of the Insurance company. Select Default if the insurance number is applicable to all the companies.

    • Is Licensed?: Specifies whether the Provider is a licensed or an unlicensed.

    • Licensed Provider: If the provider is licensed, enter the following information:
      • Provider Number: Specifies the Provider's number that is used as NPI, UPIN. 
      • Provider Number (Type): Specifies the type of the Provider's number.
    • Unlicensed Provider: If the provider is unlicensed, rendering provider information appear in place of Provider information, enter the following information:
      • Rendering Provider: Select the rendering provider whose number is used as Provider number.
      • Rendering Provider Number (Type): Specifies the type of the rendering provider number.
      • Rendering Provider Number: Specifies the number of the rendering provider.
    • Group Number: Specifies the group number.
  6. Click Save.
    The Insurance Number is added for the required provider.
