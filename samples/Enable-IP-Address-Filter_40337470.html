DrCloudEHR : Enable IP Address Filter

Created by Tejasvini Aurangabadkar on Mar 28, 2018

To enable IP Address Filtering, do the following:

  1. Click the Administration tab.
  2. On the left navigation menu, click Global Settings. The Edit Global Settings page opens.
  3. On the Security tab, select the Enable IP Address Filtering check box.
  4. Click Save.

After you enable IP Address Filtering Feature,the option becomes available at Practice tab.

  1. Click on the Practice tab.
  2. On the left navigation menu, click Administration > Facilities. The Facilities page opens.
  3. Click on the Facility hyperlink.
  4. Edit Facility page opens.
  5. Click on Access Control.
