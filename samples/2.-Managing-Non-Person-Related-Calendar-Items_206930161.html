DrCloudEHR : 2. Managing Non-Person Related Calendar Items

Created by Pankaj Bhagat on Aug 19, 2024

Scheduling a non-person specific appointment or agenda item on a calendar can be done efficiently by following these instructions.

  1. Select a location on the calendar or click on "New Event."
  2. Choose the type of appointment. Options include clinical appointments, generic events, or service appointments. For this example, select "Generic."
  3. The new event page will open. Here, you can select the event type. There will be multiple options available.
  4. If you want to schedule an ISP meeting, choose that option.
  5. Enter the location of the meeting or type it in.
  6. Add a subject for the meeting. For instance, if you meet once a month at a specific house to conduct ISP reviews, you can note that.
  7. Set the date and time for the appointment. For example, schedule it for Friday at 2 PM.
  8. Specify the duration of the meeting. You might set it to last until 5:30 PM.
  9. Assign the appointment to yourself.
  10. Set the status to "Scheduled."
  11. There is an option for a history log on the type; leave it as "None" if not needed.
  12. Type in a description for the appointment.
  13. Once all information is entered, save the appointment.


After saving, you will see the appointment listed as an ISP house review meeting, which is not tied to a specific person.
