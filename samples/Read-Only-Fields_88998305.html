DrCloudEHR : Read Only Fields

Created by Kalyan Chakravartin on Sep 11, 2020

DrCloudEHR provides an ability to display static messages and images on the form using Read only fields.

Read-only fields cannot be modified but can display some message to the end users.

The following are the read only field types:

  • Static Text Line
  • Static Large Text
  • Static Rich Text Field
  • Static Image
  • Dynamic Image Draw

Static Text Line

Static text line allows the user to display a small amount of pre-defind data on the form. 

Static Large Text

Static large text allows the user to enter a large amount of static data on the form.

Static Rich Text Field

Static rich text fields gives an ability to format the data that needs to be displayed.

System displays the data in the same pre-defined format. 

Static Image

Static image allows the user to  display a image that is still, a picture that do not move.

Dynamic Image Draw

Just like static image field, dynamic image also allows the user to display a static picture but gives a set of drawing tools for the user to draw on the image.
