DrCloudEHR : 12. Assigning a Physician to a Person

Created by Pankaj Bhagat on Aug 19, 2024

Below are the instructions for linking a physician to a person's chart.

  1. From the person's chart, select "person schedule." There is a section designated for physicians.
  2. Ensure that the physicians you want to link have already been set up in the system under the "physicians" section.
  3. To link a specific physician, navigate to the physician tab and select "new."
  4. Choose the appropriate specialization for the physician you wish to assign. For example, if you are looking for a dermatologist, select that specialization.
  5. Confirm that you are in the correct person's chart.
  6. Once in the correct chart, a dropdown menu will appear, displaying all the physicians listed in the system under the selected specialization.
  7. Select the desired physician's name from the list. For instance, you might see "Doctor Skincare" listed as a dermatologist.
  8. After selecting the physician, save the changes.
