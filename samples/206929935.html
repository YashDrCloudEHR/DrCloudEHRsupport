DrCloudEHR : 7. How to Add Medications to a Person's Chart

Created by Pankaj Bhagat, last modified on Aug 19, 2024

Adding medications to a person's chart is a straightforward process. Follow these instructions to ensure accurate documentation.

  1. From the Persons tab, select Medications.
  2. Click on "New" to open the new medication box.
  3. The system will default to the current person's chart you are in.
  4. Enter the medication name and the directions for use.
  5. The type of medication is optional but can be added if necessary.
  6. If available, input the RX number from the medication bottle.
  7. Select the physician's name from the list of physicians set up in the chart.
  8. Update the continue date once the medication is completed.
  9. For continuing a medication, enter the dosage (e.g., 300 milligrams).
  10. Specify the quantity required (e.g., 30 for a month).
  11. Set the start date to today.
  12. Optionally, enter an end date if applicable (e.g., for antibiotics).
  13. Define the interval, such as "Once a day by mouth" or "Oral."
  14. You can suggest a specific time for administration or select a time like "bedtime."
  15. Set the duration (e.g., 30 Days).
  16. Add any relevant notes or purpose for the medication.
  17. Save the information to add the medication to the chart.

To administer the medication later, select the medication and choose "Edit administered." Save the changes to complete the process.

This concludes the procedure for adding a medication into a person's chart.
