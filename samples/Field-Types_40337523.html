DrCloudEHR : Field Types

Created by Tejasvini Aurangabadkar, last modified by Kalyan Chakravartin on Sep 10, 2020

This topic contains the following subsections:

  • Basic Field Types
  • Lists
  • Patient Information
  • Read Only Fields, Visit Tools and Signatures
  • Read Only Fields
  • Visit Tools

Fields are where you enter information, or display data drawn from previous entries on forms or that is stored in the DrCloudEHR database. Fields are the functional core of every form.

For each Field Type, we’ll first show you how it appears in Form Builder Preview-mode, to help you understand the purpose of the Field from the point of view of a user of the Form. We will then show you the Field in Form Builder Edit-mode, to show you how the Field can be configured by the builder of the Form.
