DrCloudEHR : 1. Add a New Person

Created by Pankaj Bhagat, last modified on Aug 19, 2024


Steps to Add a New Person

  1. Access the Person Tab
    1. Navigate to the person tab within the EHR system.
  2. Select 'New'
    1. Click on the 'New' button to open the new person pop-up box.
  3. Enter Personal Information
    1. Full First Name: Enter the person's full first name.
    2. Title: Optionally, include a title if available.
    3. First and Last Name: Enter the first and last name separately. You can also add a middle name and an alias.
    4. Date of Birth: Input the person's date of birth.
    5. Gender: Select their gender.
  4. Additional Identifiers
    1. Social Security Number
    2. Marital Status
    3. Medicare Number (if available)
    4. Medicaid Number (if applicable)
  5. Agency and Housing Information
    1. Select Agency: Choose the relevant agency.
    2. Assign to House: If the house is set up, assign the individual to a house.
    3. House Manager: Assign a house manager if one is in the system.
  6. Optional Information
    1. Sexual Orientation
    2. Pharmacy Information: Attach if a pharmacy is set up.
    3. Referral Name, External ID, Real ID
  7. Contact and Family Information
    1. Address and Phone Number: Fill in the individual's address and phone number.
    2. Family History: Link any family member with a relevant diagnosis.
  8. Service and Medical Information
    1. ISP Dates: Enter ISP start date, intake meeting date, ISP end date, service start date, and attendance date and time.
    2. Medical Conditions: Document medical conditions, blood type, diet, and medications administered at the day program.
    3. DNR Information: If applicable, indicate if there is a Do Not Resuscitate (DNR) order and attach the URL of the documentation.
  9. Additional Demographics
    1. Monitors and Baselines: Include ethnicity and race information (multiple races can be recorded).
    2. Account Information and Special Notes: Document any special notes.
  10. Finalize and Save
    1. Ensure the individual is marked as active. Select a referral source and document any behaviors or guardianship papers. Upload a photo if desired. Click 'Save' to finalize.
  11. Review and Edit
    1. After saving, you will be directed to the new personâ€™s chart. To update any information, go to the 'Details' section and use the edit option (pencil icon).
