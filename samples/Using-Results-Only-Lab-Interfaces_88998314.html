DrCloudEHR : Using Results-Only Lab Interfaces

Created by Dan Hynes on Sep 29, 2020

Download and View Lab Test Results

DrCloudEHR automatically downloads test results from the laboratory and uploads the results to the database on a set schedule. The application parses each result and matches it with the corresponding patient, based on the matching criteria specified.

When a result is matched with a patient, the application does the following:

  • Uploads results to patient's chart. can be viewed in HTML format by clicking Practice > Procedures > Electronic Reports page.
  • Saves the results as a PDF file in the patient's Documents > Lab Report folder.
  • Sends a notification to the provider who ordered the test, and to the user group specified in the lab connection. 
  • Creates an encounter and attaches the results to it.

When a result is downloaded, the application creates a procedure order that you can use to track when the order was submitted. The procedure order might be useful when you bill the patient for the encounter.

Select Matching Criteria

You must specify the criteria DrCloudEHR uses to automatically match downloaded results with patients. You set the criteria the first time you download results from a laboratory. You can update the matching criteria as needed.

All of the criteria you specify are used to match results with patients. For example, if you select First Name, Last Name, and DOB as criteria, the application only matches results with a patient if all three items are identical in the results and in the patient demographics data.

  1. Click the Practice tab.
  2. Click Procedures in the left navigation bar, and then click Electronic Reports. 
  3. Select the checkbox for each item you want to match on in the Match Criteria box.
  4. Click Submit to save your matching criteria. 

Manually Match Test Results with a Patient

The Electronic Reports page indicates if the application is unable to match test results with a patient with the following message at the top of the page: "<number> incoming result(s) require attention". When this occurs, you can match the patient and results manually.

  1. Click the Practice tab.
  2. Click Procedures in the left navigation bar, and then click Electronic Reports. A list of reports and matched patients displays.
  3. Expand the "<number> incoming result(s) require attention" section.
  4. Click the link for each unmatched patient in the Message column. A popup window opens.
  5. Start typing patient's name in Patient field, then select the patient to match the results with.
    The application displays the Patient ID for the selected patient in the Match column.

View Test Results in PDF Format

When the application downloads lab test results, the results are saved to a PDF file to the patient's Documents > Lab Report folder.

  1. Click the Patients tab, and then select for the patient for whom you want to download the results file.
  2. Mouse over the Documents tab at the top of the Patient Summary Chart page, and then select Documents. The folder tree displays.
  3. Expand the Lab Report folder in the tree.
  4. Click the PDF file.

View Test Results Report in HTML Format

You can generate a report displaying the test results in HTML format.

  1. Click the Patients tab, and then select the patient for whom you want to view results. The Patient Summary Chart page displays.
  2. Click the Practice tab.
  3. Click Procedures in the left navigation bar, and then click Patient Results.
  4. Select the encounter the test results are saved to from the Lab Procedure Encounter menu.
  5. Click Generate Report. The test results report displays.
  6. Click Save to save the report as a PDF file to the patient's Documents > Lab Report folder.

Review and Sign Off On Results as a Provider

As a provider, you can review the lab results report for a patient. After you review the results, you electronically sign the report. The application then moves the report to the Patient Results page for the patient.

  1. Click the Patients tab, and then select the patient for whom you want to view results. The Patient Summary Chart page displays.
  2. Click the Practice tab.
  3. Click Procedures in the left navigation bar, and then click Pending Review.
    Note that you must select or create an encounter to enable the tab.
  4. Click Generate Report to view the results in HTML format in a new tab in your web browser.
  5. Return to the pending review page and select for each result from the Status menu.
  6. Select a value from the Abn menu for each result report to indicate whether the results are abnormal.
  7. After you review the report, click Sign Results.
    The report is added to the Patient Results page, which contains a list of all lab result reports for the patient.
